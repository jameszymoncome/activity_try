HTML Exercise for Input Attributes

Exercise 1:The missing attribute to specify a placeholder text of "Enter your username" for the input field is:

<p> label for="username">Username:label
    input type="text" id="username"</p>

Exercise 2:The missing attribute to set a maximum character limit of 50 for the input field is:

<p> label for="bio">Bio: /label
      input type="text" id="bio"</p>

Exercise 3: The missing attribute to make the input field required is: 

<p> label for="email">Email: /label
      input type="email" id="email"</p>

Exercise 4: The missing attribute to specify a default value of "Lovely Joyce" for the input field is:

<p>label for="name">Name:/label
      input type="text" id="name"</p>

Exercise 5:The missing attribute to disable the input field is:

<p>label for="phone">Phone:/label
      input type="text" id="phone"</p>
